# portfolio Bernard Renaud Ogodon
Ce projet est mon portfolio personnel de développeur web spécialisé en Frontend.  
Il présente une sélection de projets professionnels et académiques réalisés dans les domaines du e-commerce, des applications web et des plateformes WordPress.

L’objectif de ce portfolio est de mettre en avant mes compétences en conception d’interfaces modernes, en intégration frontend (HTML, CSS, JavaScript) ainsi qu’en développement sous WordPress / WooCommerce.

Vous y trouverez :
- Des projets WordPress
- Des applications web dynamiques
- Des solutions e-commerce personnalisées
- Une présentation claire de mes compétences techniques
